## Overview
SkinCare Builder is a web application that helps users create personalized skincare routines.  
Users answer a short skin assessment questionnaire, and the system recommends routines based on skin type, concerns, lifestyle, and personal preferences.

Users can choose natural home remedies, commercial products, or a mixed approach. The app also adjusts routines based on seasons and checks ingredients for allergies.

---

## Key Features
- Skin assessment questionnaire
- Personalized skincare routine generation
- Seasonal routine adjustments
- Ingredient safety and allergy warnings
- Save and manage multiple skincare routines
- User login and account management

---

## Tech Stack
### Frontend
- React with TypeScript
- Tailwind CSS & shadcn/ui
- Vite

### Backend
- Node.js with Express
- TypeScript

### Database
- PostgreSQL (via Neon)
- Drizzle ORM

---

## How It Works
1. User completes skin assessment
2. System analyzes skin type and concerns
3. Personalized routine is generated
4. Ingredient safety is checked
5. User can save and manage routines
